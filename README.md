# Auth0 Login Flow Challenge

This project demonstrates a simple authentication flow using Auth0.

## Features
- User login with Auth0
- Logout functionality
- Display user profile (name)

## Tech Stack
- HTML
- CSS
- JavaScript
- Auth0 SPA SDK

## How to Run
1. Clone the repository
2. Create a config.js file based on config.example.js
3. Add your Auth0 domain and clientId
4. Run a local server (e.g. python -m http.server 3000)
5. Open http://localhost:3000
